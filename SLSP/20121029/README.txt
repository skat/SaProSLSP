README for leverance 20121029
----------------------------------------------------------------------------------
Da ændringerne vedrører den fælles datamodel for R75- og R75S-servicen indeholder 
denne leverance begge services.

Nye serviceversionsnumre:

PersonKontrolOplysningHent v2.0 -> v2.1
VirksomhedKontrolOplysningHent v1.0 -> v1.1

Ændringer:

KontrolOplysningRapportIndkomstÅr er nu valgfrit i input på PersonKontrolOplysningHent. Det er fortsat obligatorisk i VirksomhedKontrolOplysningHent.

PersonKontrolOplysningHent's funktionalitetsbeskrivelse er opdateret vedr. valgfrit indkomstår.

RapportOplysningKvantumAntal er ændret til decimal uden begrænsninger (xsd:decimal)

RapportOplysningProcentProcenttal er ændret til decimal uden begrænsninger (xsd:decimal)

RapportOplysningMarkeringTekst længde er udvidet til 1000 tegn

RapportOplysningBeløbBeløb er udvidet til 15 cifre + 2 decimaler
